# 개발 명령어

## 개발 서버
```bash
npm run dev
```

## 프로덕션 빌드
```bash
npm run build
```
**중요**: 커밋 전 반드시 실행하여 빌드 성공 확인

## 프로덕션 서버 시작
```bash
npm run start
```

## 린트
```bash
npm run lint
```

## 뉴스레터 발송
```bash
npm run send-newsletter
```

## 블로그 포스트 생성
```bash
npm run generate-blog
```

## Git
```bash
git status
git add .
git commit -m "메시지"
git push
```
