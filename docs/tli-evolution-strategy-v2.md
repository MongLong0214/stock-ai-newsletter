### 1.2 예측 파이프라인 상세 (코드 기반)

**데이터 흐름:**
```
collect-and-score.ts 매일 실행
├── 6단계: snapshotPredictions() — 활성 테마 예측 스냅샷 생성
│   └── prediction_snapshots 테이블에 upsert (theme_id, snapshot_date UNIQUE)
│   └── 저장 필드: phase, confidence, risk_level, momentum, scenarios(best/median/worst)
├── 7단계: evaluatePredictions() — 14일 경과 pending 스냅샷 평가
│   └── phase_correct(boolean), peak_timing_error_days(integer) 기록
│   └── status: pending → evaluated
└── 8단계: evaluateComparisonOutcomes() — 비교 정확도 검증
    └── trajectory_correlation, stage_match → comparison_calibration 일간 요약
```

**현재 집계 정확도 계산: 존재하지 않음.**
- `phase_correct` boolean은 개별 스냅샷 레벨에서만 기록
- 전체 정확률(예: "72% 적중")을 산출하는 코드/쿼리/UI가 **어디에도 없음**
- 이전 보고서의 "16.5% 정확도" 수치는 **허위** — 산출 근거 부재

**의미:** 성적표(scorecard) 기능의 핵심 전제인 "정확도 데이터"의 존재 여부가 미확인. DB에 raw 데이터(phase_correct)는 쌓이고 있으나, 집계하여 보여줄 코드가 필요.

### 2.3 경쟁 분석 결론

**"경쟁 진공"은 과장이지만, 실질적 차별화는 존재한다.**

- **국내**: 모든 플랫폼이 "오늘의 테마 목록 + 등락률"에 머물러 있음. 핀업 ThemeLog가 가장 발전했으나 트리맵 시각화 수준이지 생명주기/점수/패턴비교는 없음
- **글로벌**: STOXX/BlackRock이 생명주기 개념을 사용하지만 기관 전용이며 리테일 제품이 아님. Morningstar는 리서치 보고서에서 테마 생명주기를 논하지만 사용자용 도구는 아님
- **TLI의 고유 조합**: 생명주기 5단계 + 정량 4요소 점수 + 과거 패턴 비교 + 예측 시나리오 → **이 4가지를 리테일 사용자에게 무료로 제공하는 서비스는 국내외 전무** (검증 기준: 14개 플랫폼 조사)

**정직한 위치:**
- 네이버증권/인포스탁이 제공하는 "테마 목록" 자체는 동일한 공개 정보
- TLI의 가치는 그 위에 올린 **분석 레이어**(점수+단계+비교+예측)에 있음
- 이 분석 레이어가 **사용자에게 실제로 유용한가**가 핵심 질문

### 3.1 법적 프레임워크

### 3.2 TLI 현재 포지션 분석

**자본시장법 §7③ 적용 요건과 TLI 부합도:**

| Safe Harbor 요건 | TLI 현황 | 충족 |
|-----------------|---------|:---:|
| 간행물/통신물 통한 조언 | 웹사이트 (통신물) | O |
| 개별성 없는 조언 | 모든 사용자에게 동일 점수/단계 표시 | O |
| 불특정 다수 대상 | 로그인 불요, 누구나 접근 | O |
| 양방향 소통 없음 | 사용자와 점수에 대한 의견 교환 채널 없음 | O |
| 무료 | "완전 무료: 영구적으로 무료로 제공" (about 페이지) | O |

**결론 (MEDIUM 신뢰도):** 현재 TLI는 §7③ safe harbor에 해당. 무료 + 비개인화 + 일방향이므로 규제 의무 없음.

**위험 요소 (향후 주의):**
1. 유료화 시 → 유사투자자문업 신고 필요 (금융위)
2. "매수 추천", "지금 투자하세요" 등 투자권유 문구 사용 시 → 투자권유 해당 가능
3. 유료 사용자와 양방향 소통 시 → 2024.8.14 개정 자본시장법에 의해 투자자문업 등록 필요

## 4. 기회 분석 — 제약조건 반영

### 4.1 제약조건

| 제약 | 의미 |
|------|------|
| 개인화 없음 | 워치리스트, 개인 알림, 맞춤 브리핑 불가 |
| 유료화 없음 | 수익 모델 불필요, 순수 제품 가치에만 집중 |
| 무료 유지 | 규제 부담 최소 (§7③ safe harbor 유지) |

### 4.2 전략적 의미

**긍정적:**
- 규제 리스크 최소화 — 가장 안전한 포지션 유지
- 사용자 진입 장벽 0 — 접근성 최대
- 유사투자자문업 신고 불필요

**제한적:**
- 개인화 없이는 "모든 사용자에게 동일한 정보" = 뉴스레터/미디어와 동일한 가치 수준
- 맞춤 알림/브리핑 불가 → 재방문 동기 약함
- DAU 성장의 핵심 레버 (관심 테마 변화 알림) 사용 불가
